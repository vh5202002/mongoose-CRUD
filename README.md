## Instructions_

#### 1. node app.js

#### 2. How to use mongoose
    https://mongoosejs.com/docs/queries.html
    
#### 3. Restart the server
    Add it to json.js
    "start": "nodemon app.js"
    nodemon app.js

#### 4. To run the app
    npm start start
